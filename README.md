
# Control MacOS Media Keys with Shortcuts

## INSTALL
```
pip3 install pyobjc-framework-Quartz
git clone https://github.com/guinmoon/ios_macos_mediakey_shortcuts
```


## USAGE:
```
python3 quartz_mediakey.py <parameter>

PARAMETERS:
-h help

-p play/pause
-n next
-r prev
-f fast
-w rewind

-u sound up
-d sound down
```
